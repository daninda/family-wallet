Сервис для учета финансов семьи "Семейная копилка"

Сведения о системе

Сервис для учета финансов семьи представляет собой автоматизированную систему, которая позволяет управлять финансами семьи: добавлять и классифицировать расходы, отслеживать их по категориям и подкатегориям, устанавливать лимиты и просматривать статистику. Пользователи могут фильтровать расходы, управлять участниками семьи и анализировать данные с помощью графиков и диаграмм.

  • Главная страница
  • Страница статистики
  • Страница управления участниками
  • Страница управления категориями и подкатегориями

Главная страница

После входа в учетную запись в приложении откроется главная страница приложения, пример которой представлен на картинке ниже.

На этой странице пользователь может выполнять следующие действия:

  • Просмотр текущих расходов: Отображаются расходы, сгруппированные по датам. Для каждого расхода показываются категория, подкатегория, стоимость, описание и время.
  • Уведомление о превышении лимита: Если общий расход превышает установленный лимит, появляется предупреждение.
  • Фильтрация расходов: Можно фильтровать расходы по стоимости, дате, категории и подкатегории. Для фильтров доступны поля для ввода диапазона стоимости и даты.
  • Выбор категории для фильтрации: Пользователь может выбрать категорию из выпадающего списка для фильтрации.
  • Навигация и действия: Есть кнопка для перехода на страницу статистики. Кнопка для добавления нового расхода.
  • Добавление нового расхода: При открытии модального окна пользователь может выбрать категорию, подкатегорию, указать описание, стоимость и дату для нового расхода.
  • Настройка фильтров: Есть возможность установить фильтры для отображения определённых расходов по стоимости и дате.

Страница статистики

На этой странице пользователь может:

  • Просматривать статистику: В зависимости от выбранного типа статистики отображается график (линейный график) или диаграмма (круговая диаграмма) с данными о тратах.
  • Выбирать тип статистики: Есть возможность переключаться между графиком и диаграммой для отображения статистики.
  • Фильтровать данные: Можно установить фильтры по дате (выбрать диапазон дат) для отображения данных в нужный период.
  • Навигация: Кнопка "Назад" для возврата на предыдущую страницу.

Страница управления участниками

Эта страница доступна только для главы семьи и на ней он может:

  • Просматривать список участников: Отображаются все участники с возможностью управления их лимитами. Для каждого участника можно изменить его лимит или удалить его из группы.
  • Управление лимитами участников: Можно установить, изменить или снять лимит для каждого участника. Доступна кнопка для снятия лимита с участника.
  • Просмотр и управление запросами на присоединение: Отображаются запросы на присоединение к группе. Можно принять или отклонить запросы на вступление.
  • Копирование кода семьи: Код семьи отображается на странице и может быть скопирован в буфер обмена.
  • Навигация: Кнопка "Назад" для перехода на предыдущую страницу.

Страница управления категория и подкатегориями

Эта страница доступна только для главы семьи и на ней он может:

  • Просматривать и управлять категориями и подкатегориями: Отображаются все категории и подкатегории. Для каждой категории и подкатегории можно удалить элемент.
  • Добавление новой категории: Открывается модальное окно для добавления новой категории. После добавления категории данные обновляются.
  • Добавление новой подкатегории: Для каждой категории можно добавить подкатегорию через модальное окно. Для подкатегории также можно указать её название и выбрать соответствующую категорию.
  • Удаление категорий и подкатегорий: Для каждой категории и подкатегории предусмотрена кнопка удаления.
  • Навигация: Есть кнопка для возвращения на главную страницу.
